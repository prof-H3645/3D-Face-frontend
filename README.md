### **使用说明**

开发使用element UI框架，github上没有node_modules这个文件夹，需要自己下载代码之后，在项目根目录下输入以下命令

```
npm install
```

如果有安装cnpm，那么用cnpm更好

有的时候只输入npm install还不够，对于我们项目来说，可能需要输入以下命令

```
npm install element-ui --save
npm install axios
```

如果还不行可以仔细看他的反馈输出，输入他提供的命令，大概就能行。
